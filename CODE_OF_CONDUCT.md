# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks are not tolerated. We value clear, evidence-led disagreement, which is the whole spirit of this project.

## Reporting

Report unacceptable behaviour to the maintainer via the contact in [SECURITY.md](SECURITY.md) or by opening a confidential issue. Reports are handled discreetly and in good faith.
